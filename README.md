**Cloud-Native Microservices Application**  
A scalable marketplace digitizing Uganda’s mitumba (second-hand clothing) trade from Owino & Kikuubo Markets.

## Tech Stack
- **Microservices**: User, Product, Order (Node.js + Express)
- **Containerization**: Docker + Kubernetes
- **Database**: MongoDB Atlas (Cloud)
- **CI/CD**: GitHub Actions
- **Monitoring**: Prometheus + Grafana
- **Innovation**: Condition Rating system + WhatsApp order sharing
